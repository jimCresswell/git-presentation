What is Git?

Git is a distributed version control system.

Once you clone a repoistory you have the full history of that repository on your computer and all branching and committing is local and fast.

Why is Git better than SVN?

  • No more difficult integration period. Creating a branch takes milliseconds. A new repository takes seconds. No more merge fear.
  • Better for offshore teams because much less network traffic.
  • Much greater visibility of the code and work in progress.
  • Easier to test small pieces of work -> “Shift Left”
  • Support for adding large file to the repository
